This project is a Java-based Student Report Card System that collects student marks, calculates totals, assigns grades, computes class statistics, and prints a structured report.
It is designed as a console application and demonstrates beginner-to-intermediate Java concepts such as:
 -Arrays
 -Loops
 -Conditional statements
 -Iput handling using Scanner
 -Formatted output
 -Basic data processing
